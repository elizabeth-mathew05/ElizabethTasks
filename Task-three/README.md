# Income Expense Calculator

A simple, responsive web app to track your income and expenses. Add, edit, delete entries, filter by type, and see totals at a glance. Data is saved in your browser.

## Features
- Add, edit, and delete income/expense entries
- Filter by all, income, or expense
- See total income, expense, and balance at the top
- Reset button to clear all entries
- Clear Fields button to clear only input fields
- Data persists via localStorage
- Responsive design for desktop and mobile

## Usage
1. Enter a description, amount, and select type (income/expense).
2. Click Add to save the entry.
3. Use Edit/Delete buttons to modify or remove entries.
4. Use filters to view only income, expense, or all entries.
5. Click Reset All to clear all entries, or Clear Fields to reset the form.

## Tech
- HTML, CSS (Tailwind), JavaScript
- SVG icons for edit/delete in assets/
