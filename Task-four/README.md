# Memory Game

A simple, responsive memory game built with HTML, CSS, and JavaScript.

## How to Play
- Click on cards to flip them and reveal the symbol.
- Find matching pairs. If two cards match, they stay face-up. If not, they flip back.
- Try to match all pairs with as few moves as possible!
- Use the Restart button to reset and shuffle the board.

## Features
- Card flipping and matching logic
- Shuffle function for randomizing cards
- Responsive design for desktop and mobile
- Clean, well-documented code

## Setup
Just open index.html in your browser. No installation required.
