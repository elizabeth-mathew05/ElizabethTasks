# Memory Card Match Game

A simple, responsive memory game built with HTML, CSS, and JavaScript.

## How to Play
- Click one card to flip and reveal its front face (emoji symbol).
- Click a second card to try to make a matching pair.
- If both cards match, they remain face-up.
- If they do not match, they flip back after a short delay.
- Match all pairs to finish the game.
- Use the Restart button anytime to reset and reshuffle the board.

## Features
- Card flipping and matching logic
- Shuffle function for randomizing cards
- State tracking for flipped cards and matched pairs
- Responsive design for desktop and mobile
- Clean, well-documented code

## Setup
Open index.html in your browser. No installation required.
